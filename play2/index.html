Playframewrok2系 + AngularJS1系 on Heroku

概要

Webサイトを以下の構成で作るPJの雛形構築

  • サーバサイド: Playframework2系でREST API
  • クライアントサイド: AngularJS1系
  • プラットフォーム: Heroku

環境

  • OS X ElCapitan 10.11.4

prodモードで動かすため、アプリケーションシークレットを環境変数 からの取得に設定する。

# applicaton.confに追記
play.crypto.secret = ${?APPLICATION_SECRET}
